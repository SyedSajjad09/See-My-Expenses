# Expense Tracker

[See My Expenses](https://see-my-expenses.netlify.app/) is a simple web application to help track daily expenses. It allows users to log their expenses, categorize them, and get an overview of their spending.

## Features

- Add and categorize expenses
- View a list of all expenses
- Simple, responsive interface

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Hosting: Netlify

In short: This is just a simple project where you can see or calculate your budget and expenses by listing the items and prices.
